# 05. Portal Web Responsivo (Angular)

Desarrollo frontend dinámico basado en las prácticas del Bootcamp Full-Stack Web.

## 🎯 Contexto del Negocio
Una buena API necesita una interfaz de usuario que la consuma eficientemente. El objetivo era desarrollar una aplicación web de página única (SPA) reactiva y totalmente responsiva, orientada a ofrecer una experiencia de usuario (UX) fluida.

## 🛠️ Arquitectura y Stack
*   **Framework:** Angular, TypeScript.
*   **Estilos:** HTML5, CSS3, SCSS / Bootstrap.
*   **Arquitectura:** Basada en componentes reutilizables y servicios inyectables.

## 🧠 Retos Técnicos
*   **Gestión del Estado:** Mantener el estado de la aplicación coherente a través de múltiples componentes.
*   **Llamadas Asíncronas:** Consumir servicios RESTful (API backend) utilizando RxJS y Observables, gestionando los tiempos de carga y los errores de red.
*   **Diseño Responsivo:** Garantizar que la interfaz se adaptara perfectamente a pantallas móviles y de escritorio (Mobile-First).

## 📈 Impacto y Resultados
*   *(Sección en edición)* Una aplicación web ágil que elimina las recargas de página, optimizando la experiencia final.
