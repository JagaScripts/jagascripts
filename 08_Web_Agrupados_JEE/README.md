# 08. Web Agrupados (Java 2 EE)

Aplicación web empresarial multiplataforma construida sobre el ecosistema clásico Java Enterprise Edition.

## 🎯 Contexto del Negocio
Plataforma orientada a la agrupación de ofertas y cupones de diferentes comercios (tipo Groupon). El negocio requería un backend robusto capaz de manejar sesiones concurrentes de múltiples tipos de usuarios (Administradores, Empresas, Clientes), procesar transacciones y renderizar vistas dinámicas directamente desde el servidor.

## 🛠️ Arquitectura y Stack
*   **Frameworks:** Java EE, Servlets, JSP (JavaServer Pages).
*   **Frontend Components:** PrimeFaces, JSF (JavaServer Faces).
*   **Servicios Externos:** Integración con la API de Google Maps para geolocalización.

## 🧠 Retos Técnicos
*   **Ciclo de vida JSF:** Comprender y dominar el complejo ciclo de vida de los componentes JSF/PrimeFaces para la recarga dinámica de la interfaz mediante AJAX.
*   **Integración de APIs y Seguridad:** Incorporar mapas interactivos (Google Maps) inyectando coordenadas dinámicamente desde el backend Java al frontend XHTML de forma segura.
*   **Gestión de Sesiones:** Controlar los roles de usuario y los estados de la sesión HTTP en un entorno de servidor.
