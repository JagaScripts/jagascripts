La empresa AGRUPADOS S.A. El objetivo de la empresa sistema de búsqueda de ofertas, un módulo de estadísticas de las ventas para los negocios y construyas una API de consulta a través de servicios web (REST) para poder acceder a la plataforma y consultar datos desde dispositivos móviles. Este sistema deberá permitir la consulta por localización y simular el proceso de compra de cupones de ofertas (si deberá registrar la operación pero no realizar el pago). Si un usuario quiere comprar un cupón de oferta a través de un método de la API, se deberá registrar la compra con un código autogenerado y el periodo de validez para que se pueda comprobar que es una oferta válida a través de la web.
